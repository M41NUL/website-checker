## 🚀 Quick Install

```bash
git clone https://github.com/M41NUL/website-checker
cd website-checker
```

```bash
pip install rich pyfiglet requests python-whois pyOpenSSL
```

```bash
python website_checker.py
```

```bash
# Bash version
chmod +x website_checker.sh
./website_checker.sh
```
